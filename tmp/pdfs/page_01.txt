STS Project • Sprint 1
What Sprint 1 delivered
Anonymous users can access the welcome
page and register an account.
Registered users can log in and
authenticated users can log out.
The app foundation is in place: MVC routing,
Identity-based auth, EF Core data access,
and MySQL configuration.
Team ownership
Viktor: Welcome screen + user registration
Nikolay: User login + user logout
ASP.NET Core MVC app with HomeController and
AccountController endpoints.
ApplicationUser extends ASP.NET Identity and is stored
through AppDbContext.
Program.cs wires Identity, EF Core, MySQL, custom
password validation, cookies, and routing.
The project also includes 23 automated tests covering
homepage, registration, login, logout, and validation
behavior.
